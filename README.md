# CrypteX

Образовательная игра на Unity, обучающая основам криптографии через серию мини-игр по взлому/декодированию шифров. Дипломный проект.

## Стек

- **Движок:** Unity, C#
- **UI:** Unity UI Toolkit (UI Elements)
- **Сохранение прогресса:** локальная конфигурация в JSON (`Application.persistentDataPath`)

## Игровая механика

Игрок проходит уровни, каждый из которых построен вокруг отдельного шифра или криптографического алгоритма. Чтобы пройти уровень, нужно решить загадку — расшифровать сообщение, подобрав ключ или восстановив исходный текст с помощью игровой механики, демонстрирующей принцип работы алгоритма.

### Реализованные шифры и алгоритмы (мини-игры)

- Шифр Цезаря (Caesar)
- Атбаш (Atbash)
- Шифр Виженера (Vigenère)
- Шифр "Изгородь" (Rail Fence)
- ROT13
- Полибианский квадрат (Polybius)
- Азбука Морзе (Morse code)
- Одноразовый блокнот / шифр Вернама (Vernam / one-time pad)
- XOR-шифрование
- MD5 (демонстрация хеширования)
- MAC (код аутентификации сообщения)
- RSA (асимметричное шифрование — подбор параметров p, q, φ, d)
- S-DES (упрощённый DES)
- Схема Шамира (разделение секрета)

Для некоторых шифров (Caesar, Atbash) дополнительно реализованы демонстрационные контроллеры, наглядно показывающие принцип шифрования/дешифрования.

## Архитектура

```
Assets/Scripts/
├── Managers/
│   ├── GameManager        # состояние игры (меню, игра, пауза), сохранение конфигурации
│   ├── LevelManager        # система уровней, загадки (шифротекст/ответ/ключ) для каждого уровня
│   ├── DialogueManager      # система диалогов
│   ├── UIManager            # управление UI-экранами
│   └── AudioManager          # звук и музыка
├── Gameplay/
│   ├── *HackMinigame         # логика мини-игр для каждого шифра/алгоритма
│   └── *DemoController        # демонстрационные контроллеры (Caesar, Atbash)
├── UI/
│   ├── ScreenInitializers/    # инициализация экранов (главное меню, выбор уровня, диалог, пауза, конец игры)
│   └── Util/                  # утилиты UI (подсветка элементов, FPS-счётчик, чит-консоль)
└── Dialogues/
    └── DialogueData            # данные диалогов
```

Каждый уровень описывается структурой `RiddleData` (шифротекст, ожидаемый расшифрованный текст, ключ) и связан с соответствующей UI-сценой мини-игры через `LevelManager`.

## Запуск проекта

### Требования

- Unity (версия указана в `ProjectSettings/ProjectVersion.txt`)

### Открытие проекта

1. Откройте Unity Hub
2. Добавьте папку проекта (`Add` → укажите корень репозитория)
3. Откройте проект — Unity автоматически импортирует ассеты
4. Запустите сцену из `Assets/Scenes` (главное меню) в редакторе

## Примечания

Проект разработан в качестве дипломной работы по теме «Разработка игрового симулятора киберрасследования на платформе Unity для демонстрации принципов работы классических криптографических алгоритмов». Алгоритмы реализованы в учебных целях для демонстрации принципов работы, а не как криптографически безопасные реализации для продакшена.
